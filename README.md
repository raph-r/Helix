# Helix
A third-person shooter game.
If you have any question, please, let me know.

## Engine
* Unity

## Features
* Camera animations
* Helicopter animations
* Bombs take and deal damage

## Controls

* Space - Shoot
* , - Rotate to left
* . - Rotate to right
* W - Forward
* S - Backward
* A - Left
* D - Right
* PageUP - Adds height
* PageDown - Subtracts height
* 1 - Third person camera
* 2 - First person camera
* 3 - TopDown camera
